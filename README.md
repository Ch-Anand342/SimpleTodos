Simple Todos

A React.js Todo application that helps users manage their daily tasks efficiently. This enhanced version allows users to add, edit, delete, and mark todos as completed, along with support for creating multiple todos at once.

Features
View a list of todos
Add new todos
Delete existing todos
Edit todo titles
Save updated todo titles
Mark todos as completed using a checkbox
Completed todos are displayed with a strike-through effect
Add multiple todos with the same title by entering the title followed by the number of todos (Example: Buy Milk 5)
Responsive and clean user interface
Technologies Used
React.js
JavaScript (ES6)
HTML5
CSS3

Installation
Clone the repository
git clone <repository-url>
Navigate to the project directory
cd enhancementOfSimpleTodos
Install dependencies
npm install
Start the development server
npm start

The application will run at:

http://localhost:3000
Usage
Enter a todo title and click Add to create a new task.
To create multiple todos with the same title, enter the title followed by a number.

Example:

Read Book 3

This creates three todos with the title Read Book.

Click Edit to modify a todo.
Click Save to save the updated title.
Click the checkbox to mark a todo as completed.
Click Delete to remove a todo.
